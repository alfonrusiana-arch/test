MUNICIPALITY OF BUUG OFFICIAL WEBSITE

PROJECT STRUCTURE:
------------------
/ (root)
├── index.html          - Homepage
├── about.html         - About Buug page
├── tourism.html       - Tourism attractions page
├── government.html    - Local government page
├── services.html      - Municipal services page
├── contact.html       - Contact page
├── css/
│   └── style.css     - Main stylesheet
├── js/

FEATURES:
---------
✓ Mobile-responsive design
✓ 6 navigation menus (Home, About, Tourism, Government, Services, Contact)

✓ Smooth animations and transitions
✓ Interactive elements
✓ Contact form with validation
✓ Scroll to top button
✓ Image lazy loading
✓ Counter animations for statistics
✓ Social media links
✓ Google Maps integration
